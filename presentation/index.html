Compliance with the Road Traffic Code in an Autonomous Driving Agent

Stop Sign Recognition and Traffic Light Detection

Report Presentation Project

This is a project on the course Autonomous Agents that enhances an existing autonomous driving agent by ensuring compliance with the road traffic code through stop sign detection and traffic light recognition.

Instructions:

  1. Install WeBots R2023b
  2. Open the world "city.wbt"
  3. Compile the four controllers
  4. Run and observe the results
